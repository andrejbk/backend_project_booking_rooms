BookingPro

Developed a RESTful API for a hotel room booking platform

About the project

Designed and developed a Python backend application using the asynchronous FastAPI framework with PostgreSQL as the primary database. Implemented robust data validation, serialization, and schema management with Pydantic, and managed database access through SQLAlchemy ORM. Integrated Redis for high-performance caching and Celery for asynchronous background task processing. Ensured code quality through comprehensive testing with Pytest and static analysis, linting, and formatting with Ruff. Containerized the application using Docker and deployed it to a Linux VPS. Established an automated GitLab CI/CD pipeline to support continuous integration and deployment. Configured a custom domain and set up Nginx as a reverse proxy to handle incoming HTTP requests, SSL termination, and request routing to the application services.
